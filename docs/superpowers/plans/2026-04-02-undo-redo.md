# Undo/Redo for Mask — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add 20-step undo/redo to the paintable mask — one step per stroke or reset — with Cmd/Ctrl+Z keyboard shortcuts and Undo/Redo buttons.

**Architecture:** All history logic lives inside `useMask.js`. A `historyRef` array holds `Uint8Array` snapshots (max 20). A `historyIndexRef` tracks the current position. A `historyVersion` state counter triggers re-renders when history changes so `canUndo`/`canRedo` update correctly. `App.jsx` adds two buttons and reads the four new return values.

**Tech Stack:** React hooks, WebGL 2 (existing), no new dependencies

- [ ] **Step 2: Add history refs and historyVersion state**

At the top of the `useMask` function body, directly after the existing `isPaintingRef` line, add:

```js
  const MAX_HISTORY     = 20;
  const historyRef      = useRef([]);
  const historyIndexRef = useRef(-1);
  const [historyVersion, setHistoryVersion] = useState(0);
```

- [ ] **Step 3: Add saveSnapshot callback**

Add this after the existing `uploadMask` callback (after its closing `}, [canvasRef]);`):

```js
  const saveSnapshot = useCallback(() => {
    if (!maskArrayRef.current) return;
    historyRef.current = historyRef.current.slice(0, historyIndexRef.current + 1);
    historyRef.current.push(maskArrayRef.current.slice());
    if (historyRef.current.length > MAX_HISTORY) historyRef.current.shift();
    historyIndexRef.current = historyRef.current.length - 1;
    setHistoryVersion(v => v + 1);
  }, []);
```

- [ ] **Step 4: Add undo and redo callbacks**

Add these after `saveSnapshot`:

```js
  const undo = useCallback(() => {
    if (historyIndexRef.current <= 0) return;
    historyIndexRef.current--;
    maskArrayRef.current.set(historyRef.current[historyIndexRef.current]);
    uploadMask();
    setHistoryVersion(v => v + 1);
  }, [uploadMask]);

  const redo = useCallback(() => {
    if (historyIndexRef.current >= historyRef.current.length - 1) return;
    historyIndexRef.current++;
    maskArrayRef.current.set(historyRef.current[historyIndexRef.current]);
    uploadMask();
    setHistoryVersion(v => v + 1);
  }, [uploadMask]);
```

- [ ] **Step 5: Derive canUndo / canRedo**

Add these two lines immediately after the `redo` callback. They are plain variables (not state) — they read the latest ref values each render, triggered by `historyVersion` changes:

```js
  const canUndo = historyIndexRef.current > 0;
  const canRedo = historyIndexRef.current < historyRef.current.length - 1;
```

- [ ] **Step 6: Clear history on mask resize**

- [ ] **Step 8: Save snapshot before resetMask clears**

Find the existing `resetMask` callback:

Replace it with:


- [ ] **Step 9: Add keyboard shortcuts effect**

Add this new `useEffect` after the pointer events effect (before `resetMask`):

```js
  useEffect(() => {
    const onKeyDown = (e) => {
      const mod = e.metaKey || e.ctrlKey;
      if (mod && !e.shiftKey && e.key === 'z') { e.preventDefault(); undo(); }
      if (mod &&  e.shiftKey && e.key === 'z') { e.preventDefault(); redo(); }
    };
    window.addEventListener('keydown', onKeyDown);
    return () => window.removeEventListener('keydown', onKeyDown);
  }, [undo, redo]);
```
